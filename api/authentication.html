# Autenticação

A API da PHX Cargo utiliza tecnologia OAuth2 de modo a TOKENIZAR as requisições mantendo o controle de acesso de nossos clientes e parceiros;

Para autenticar na API da PHX Cargo é necessário seguir as instruções abaixo:

# GET TOKEN
