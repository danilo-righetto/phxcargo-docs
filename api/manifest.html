# Manifest

O que é um manifesto?: Um manifesto é um documento (XML) onde será MANIFESTADO todas as informações que um MASTER e suas respectivas remessas das quais o SISCOMEX pede ao importar ou exportar uma carga.

# API - Manifest

Através da nossa API com autenticação é possível realizar diversas operações sobre os manifestos de cada cliente.

# GET ALL MANIFESTS

Podemos por essa API recuperar todos os manifestos (do usuário que está logado pela API).

Autenticação

REQUEST

https://api.phxcargo.net/manifests
