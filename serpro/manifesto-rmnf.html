# Manifesto - Envio e Retorno

# Gerar manifesto

Para gerar o manifesto de carga via sistema, é necessário seguir um conjunto de regras definidas pelo Siscomex/SERPRO para que a transmissão seja executada com sucesso.

# Campos obrigatórios

  • uaDespacho
  • paisOrigem
  • numeroManifesto - Caso o arquivo seja uma retificação
  • tipoXML
  • hashManifesto
  • sequenciaManifesto - Caso o arquivo seja uma retificação esse campo deve ter o valor nulo
  • modalidade
  • dtHorarioEnvioArquivo
  • cnpj
  • numeroRemessa
  • numeroMaster
  • valorTotal
  • moedaValorTotal
  • descricao
  • peso
  • frete
  • moedaFrete
  • volumes
  • freteModoPagto
  • tipoDocumento
  • nome

# Envio

O nome do arquivo de exemplo é o K3244.K07610XA.PEX00044.D200710.H090752 e sua nomenclatura influencia na identificação e processamentos dos dados.

Onde:

K3244 significa envio de transmissão

K07610XA significa Produção

K07610XA significa que o tipo do arquivo é XML

K07610XA significa que o tipo do arquivo enviado do SISCOMEX A - Manifesto de Carga

PEX00044 onde PEX = Código da empresa que está enviando os arquivos + (PEX00044) sequencial do arquivo enviado.

D200710 onde D é fixo + aammdd (ano, mês e dia)

H090752 onde H é fixo + hhmmss (hora, minuto e segundo)

# Retorno manifesto

É através do arquivo de retorno do manifesto que conseguimos obter o numeroManifesto pelo Siscomex e com esse numeroManifesto devemos utilizá-lo até o fim da operação.

# Campos obrigatório do SISCOMEX

O Siscomex deve de maneira obrigatório enviar os campos abaixo no arquivo de retorno:

  • tipoXML
  • cnpj
  • dataHorarioEnvioArquivo
  • dataHorarioRegistroManifesto
  • sequenciaManifesto
  • numeroManifesto
  • numeroManifesto
  • codigoMensagem
  • mensagem - Caso haja algum erro no envio do manifesto

# Recebimento

O nome do arquivo de exemplo é o K3249.K07610XA.PEX00044.D200710.H100750 e sua nomenclatura influencia na identificação e processamentos dos dados.

Onde:

K3249 significa retorno de transmissão

K07610XA significa Produção

K07610XA significa que o tipo do arquivo é XML

K07610XA significa que o tipo do arquivo enviado do SISCOMEX A - Retorno do Manifesto de Carga

PEX00044 onde PEX = Código da empresa que está enviando os arquivos + (PEX00044) sequencial do arquivo enviado.

D200710 onde D é fixo + aammdd (ano, mês e dia)

H100750 onde H é fixo + hhmmss (hora, minuto e segundo)

O modelo do arquivo de arquivoRetornoManifesto pode ser visto abaixo:

retorno do manifesto 
<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
<arquivoRetornoManifesto xmlns="http://www.siscomex.gov.br/remessa/v1">
    <transmissao>
        <tipoXML>RMNF</tipoXML>
    </transmissao>
    <declarante>
        <empresa>
            <cnpj>10257602000182</cnpj>
        </empresa>
    </declarante>
    <manifesto>
        <dataHorarioEnvioArquivo>2020-07-10T10:33</dataHorarioEnvioArquivo>
        <dataHorarioRegistroManifesto>2020-07-10T09:34</dataHorarioRegistroManifesto>
        <sequenciaManifesto>00044</sequenciaManifesto>
        <numeroManifesto>PEX202007092999</numeroManifesto>
    </manifesto>
</arquivoRetornoManifesto>

Observação: O manifesto gerado pelo sistema não inclui a Presença de Carga na modalidade expressa (Courier).
